Splitflapdisplay

Drive a set of Krone split-flap displays with ATmegas

Download this project as a .zip file Download this project as a tar.gz file

Split-flap displays are found in train stations and airports.

I've managed to come across a couple of salvaged displays originally manufactured by Krone (and I suspect based on an AEG design), and have finally gotten around building the necessary hardware. I will fill this wiki with information about them soon, until then enjoy a couple of YouTube videos of my experiments:
